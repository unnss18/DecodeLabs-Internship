# Smart Environmental Monitor

## Overview

This project interfaces a DHT11 temperature and humidity sensor with an Arduino Uno.

The sensor data is collected every 2 seconds and displayed on the Arduino Serial Monitor.

## Components Used

- Arduino Uno
- DHT11 Sensor
- Jumper Wires
- USB Cable

## Connections

| DHT11 | Arduino Uno |
|--------|-------------|
| VCC | 5V |
| DATA | Pin 2 |
| GND | GND |

## Output

Temperature: 29°C

Humidity: 63%
